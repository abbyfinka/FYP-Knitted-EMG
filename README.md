# Development of a Flexible EMG Readout Board for Wearable Hand Movement Prediction

Final Year Project - MEng Electrical and Electronic Engineering - Imperial College London

Supervisor: Professor Kristel Fobelets

## Repository Structure
This repo contains:
- `\hardware`: Contains PCB design files
- `\App`: Python code to run GUI to visualise the signals
- `\ESP32`: Firmware to run on ESP32 to acquire signals
- `FinalReport.pdf`: Final Report, Chapter 9 User Guide may be helpful if trying to use this software
